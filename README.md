# cepheid-warp-mapper

Collaborative project to map the Milky Way's warp using Cepheid variable stars.

## Goal
To use a catalog of Cepheid variables to map the three-dimensional structure of the Milky Way's disc and quantify its warp.

## Project Structure
- `scripts/` – analysis code
- `data/` – Cepheid catalog (CSV files)
- `figures/` – output plots

## Collaboration Workflow
All code changes must be submitted via a Pull Request for review before being merged into the `main` branch. This ensures proper code review and collaboration.

1. Fork the repository (optional).
2. Create a branch for your task.
3. Implement your changes.
4. Submit a Pull Request (PR) describing the changes.
5. Request review from the mentor or other team members.
6. Address any review comments.
7. Once approved, the PR can be merged into `main`.

Refer to the GitHub Issues for specific tasks and assignments.
